# CS Degree Portfolio

A collection of academic projects and coursework from my Computer Science degree, documenting some of the code I wrote along the way.

## Repository Structure

* **`cs210-cpp-investment-calculator/`**
  * A C++ command-line application that projects investment balances over time, calculating compounding interest with and without additional monthly deposits.
* **`cs300-course-planner/`**
  * A course management directory program that utilizes a Binary Search Tree (BST) data structure to efficiently parse, sort, and search through academic courses and their prerequisites.
